# BSFF Генератор

Генератор протоколов и читалка для BSFF формата.
Вдохновлено Нейрочисткой.

## Возможности

- Генерация протоколов в формате Markdown
- Чтение протоколов из MD и DOCX файлов
- Темная и светлая темы
- Настройка шрифта/размера/прогресс-бара чтения
- Сохранение данных в LocalStorage

## Установка локально

```bash
npm install
```

## Запуск

```bash
npm start
```

Сайт будет доступен по адресу `http://localhost:3000`
